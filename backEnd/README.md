# Backend Code Structure

The `backEnd` folder contains the core backend code for the project. It is organized into subfolders based on the application’s architecture. Each folder has a specific purpose to ensure clarity and modularity.

### Folder Overview
- **controller/**: Handles HTTP requests and responses.
- **service/**: Contains core business logic.
- **dao/**: Interacts with the database.
- **model/**: Defines data structures.
- **util/**: Provides utility methods for common tasks.
- **config/**: Manages application configuration settings.
